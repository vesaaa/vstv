# 更新日志（VsTV / vstv）

应用展示名 **VsTV**；仓库与构建标识为 **vstv**。Release 标签与 `versionName` 对齐（当前主线 **`0.1.x`** / **`v0.1.x`**）。**自 0.1.9 起随每次发版持续追加本节记录**；更早年份可结合 Git 历史与 Release 附件查看。

## [0.1.10] - 2026-03-30

- **设置 · 应用更新**：`_isUpdating` 改为 `mutableStateOf` 并在主线程更新，修复确认更新后设置页不显示「下载中」/连接与进度的问题；`finally` 中在主线程清除下载状态；重复点击时的 Toast 附带当前百分比。

## [0.1.9] - 2026-03-30

- **设置**：主页分类顺序调整，**「关于」移至最后一格**；原「更多设置」中的 **设置页面地址**、**切换扫码用的本机 IP** 与 **二维码** 并入 **「关于」** 详情；**移除「更多设置」** 分类与 `SettingsCategoryMore`。
- **文档**：`CHANGELOG.md` 不再只保留两个版本条目，后续版本均在此累积更新说明。

## [0.1.8] - 2026-03-30

- **设置 · 应用更新**：下载与安装状态集中到「应用更新」列表项（副标题显示连接/进度百分比，右侧摘要显示「下载中」「n%」「安装」等）；减少与列表重复的 Toast；下载中或正在打开安装界面时短按该项不再触发重新检查更新。

## [0.1.7] - 2026-03-30

- **文档**：`CHANGELOG.md` 曾精简为仅保留少量条目（现已改为持续维护，见上）。

## [0.1.6] - 2026-03-30

- **在线更新**：部分设备会话安装误报 `INSTALL_FAILED_ALREADY_EXISTS` 时自动改走系统安装界面；优化相关说明文案。
- **（0.1.5）播放页底部菜单**：`TvLazyRow` 左侧 `contentPadding`，避免首项圆角被裁切。
