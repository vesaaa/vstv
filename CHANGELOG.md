# 更新日志（VsTV / vstv）

应用展示名 **VsTV**；仓库与构建标识为 **vstv**。Release 标签与 `versionName` 对齐（当前主线 **`0.1.x`** / **`v0.1.x`**）。更早历史见 Git 提交与 Release 附件说明。

## [0.1.7] - 2026-03-30

- **文档**：`CHANGELOG.md` 重写，删除早前冗长记录；自本版起仅维护 **最近两个版本** 的条目，更早变更见 Git / Release。

## [0.1.6] - 2026-03-30

- **在线更新**：部分电视 / 定制系统在应用内 **PackageInstaller 会话安装** 覆盖本应用时，会误报 `INSTALL_FAILED_ALREADY_EXISTS`（-1）。此时自动 **改走系统安装界面**（与手动打开 APK 一致），并提示「已改为系统安装界面」；成功完成会话安装时清理内部状态，避免串单。
- **说明文案**：上述错误场景下，不再使用易误解的「会话安装器与本地安装器不一致」表述；若无法自动回退（如安装包已删），提示改为更直白的排查建议。
- **（0.1.5）播放页底部菜单**：`TvLazyRow` 增加左侧 `contentPadding`，避免首项「节目单」贴在列表左缘时左侧圆角 / 焦点环被视口裁切。
